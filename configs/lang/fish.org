* Install Fish

  To get start with [[https://fishshell.com/][Fish]] shell, we need to install it.

  The method to install Fish depends on your platform, mine use Mac OSX and Gentoo Linux.

** Mac OSX

   To install [[https://fishshell.com/][Fish]] on Mac OSX, just use [[http://brew.sh/][homebrew]] to achive it

   : brew install fish

   After install, you need to add following line to =/etc/shells= for chsh it

   #+BEGIN_SRC sh
     /usr/local/bin/fish
   #+END_SRC

   If you want to set it as default shell, juse type

   : chsh -s /usr/local/bin/fish

** Gentoo Linux

   To install [[https://fishshell.com/][Fish]] on Gentoo Linux, just use [[https://wiki.gentoo.org/wiki/Portage][portage]] to build it

   : emerge app-shells/fish

   If you want to set it as default shell, juse type

   : chsh -s /bin/fish
